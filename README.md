## About Me 🤖

I'm an MLOps and Machine Learning engineer based in Lagos 🇳🇬, living at the intersection of data science and software engineering. My work spans the full ML lifecycle — from feature engineering and experiment tracking to containerised serving, orchestration, and cloud infrastructure.

I believe a well-designed pipeline is as important as a well-tuned model. If it can't be reproduced, monitored, and redeployed without drama, it's not done yet.

- 🔭 Currently building: **end-to-end ML pipelines with automated retraining, drift detection, and monitoring**
- 🌱 Learning: **LLMOps, distributed training, and feature store architecture**
- 💬 Ask me about: **MLflow · Kubeflow · DVC · Airflow · FastAPI · Docker · Kubernetes · SageMaker · Terraform · CI/CD for ML**
- ⚡ Fun fact: I've debugged more broken Airflow DAGs than I've had hot meals — the DAGs usually win round one
